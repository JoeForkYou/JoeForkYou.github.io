1 STL 简介

STL是Standard Template Library的简称，中文名标准模板库
STL可分为
容器(containers)、
迭代器(iterators)、
空间配置器(allocator)、
配接器(adapters)、
算法(algorithms)、
仿函数(functors)六个部分。

选自百度词条STL百度词条
C++标准中，STL组件被组织命名为以下13个头文件

< algorithm>
< deque>
< functional>
< iterator>
< vector>
< list>
< map>
< memory.h>
< numeric>
< queue>
< set>
< stack>
< utility>
